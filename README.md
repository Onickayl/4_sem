# Snake

Игра змейка.

000000>

Твоя задача есть красных кроликов "X" и расти. Учти, что столкновение с самим собой, с другими змейками или с преградами влечёт за собой неминуемую смерть... 

В игре есть бонусы в виде магических атрибутов, которые появляются на поле. В зависимости от того какой магический премдмет твоя змейка слопает, у неё появится суперсила или нет :)

Магические атрибуты:
- жёлтое яблоко "@" даёт увеличение змейки на 3 дольки
- синяя снежинка "*" отбирает 5 долек у змейки 

Если на поле не осталось змеек, то игра автоматически завершается - gameover.

Управлять змекой можно с помощью стрелок или WASD.

Удачи!
